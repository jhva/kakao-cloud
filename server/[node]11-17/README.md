# Nodejs

### module
- 독자적으로 실행가능한 단위 
    - 함수
    - 클래스 

### Package
- 1개 이상의 파일로 구성된 배포단위

### Process
- 실행 중인 프로그램

### Thread
- 독자적으로 실행할 수 없는 Process 작업 단위 

### NPM(Node Package Manage)
> 노드에서의 패키지 매니저인데 지금은 거의 모든 자바스크립트 라이브러리들이<br/>
저장소에 있기 때문에 자바스크립트 라이브러리들은 거의 모두 NPM을 이용해서 사용

- package.json
    - 노드에서 패키지 관리를 위한설정 파일
